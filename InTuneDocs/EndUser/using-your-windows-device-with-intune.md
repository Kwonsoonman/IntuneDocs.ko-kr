# Windows デバイスを Intune で使用する

この記事には、エンドユーザーが Intune 管理対象デバイスで実行できるすべてのタスクへのリンクが掲載されています。 IT 管理者はこの情報を参照して、ユーザーがさまざまなタスクを実行するための手順を確認できます。タスクはカテゴリ別にグループ化されています。 
- ポータル サイト アプリのインストールと Intune の登録 
- 登録後の管理タスク
- ポリシー準拠の問題
