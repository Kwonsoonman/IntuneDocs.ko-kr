# 장치가 최대 운영 체제 버전에 부합되지 않는 경우

IT 관리자가 Windows 장치에서 허용하지 않은 운영 체제의 최대 버전을 설정했지만 장치가 해당 설정을 준수하지 않을 수 있습니다. 장치에 허용되는 최대 운영 체제를 확인하려면 IT 관리자에게 문의해야 합니다.
