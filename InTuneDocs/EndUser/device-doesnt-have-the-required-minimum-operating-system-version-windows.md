# 장치에 필요한 최소 운영 체제 버전이 없는 경우

Windows 장치가 IT 부서에서 필요로 하는 최소 운영 체제 버전을 충족하지 않습니다. 모바일 장치를 업데이트하기 전에 **설정** &gt; **정보**를 선택하여 현재 운영 체제 버전을 확인합니다. 그런 다음 장치를 업데이트하려면 **설정** &gt; **전화 업데이트** &gt; **업데이트 확인**을 선택합니다.

컴퓨터를 업데이트하기 전에 **운영 체제**를 검색하고 프롬프트에 따라 현재 운영 체제 버전을 식별합니다. Windows 10 이전 운영 체제가 설치된 컴퓨터를 업데이트하려면 **Windows 업데이트**를 검색합니다.

Windows 10 및 Windows 8.1에 대한 버전을 확인하려면 Windows 로고 키 + R을 선택하고, 대화 상자에서 **winver**를 입력하고 Enter를 선택합니다.

- Windows 10의 경우 버전 번호를 무시하고 OS 빌드 번호를 확인합니다(예: OS 빌드 10586.29). 10586.29에서 "10"은 Windows 10을 나타냅니다.
- Windows 8.1의 경우 버전 번호를 확인합니다. 8.1 대신 6.3.*xxxx*가 표시됩니다.

여전히 도움이 필요하세요? IT 관리자에게 문의하세요. IT 관리자의 연락처 정보는 [회사 포털 웹 사이트](http://portal.manage.microsoft.com)를 참조하세요.
