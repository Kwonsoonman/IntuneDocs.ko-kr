# iOS 장치가 최신 운영 체제 버전 설정에 부합되지 않는 경우

iOS 장치가 사용할 수 있는 최신 운영 체제 버전에 대한 IT 관리자의 정책을 준수하지 않습니다. IT 관리자에게 문의하세요. IT 관리자의 연락처 정보는 [회사 포털 웹 사이트](http://portal.manage.microsoft.com)를 참조하세요.
