# <a name="your-macos-devices-operating-system-version-isnt-yet-supported"></a>사용 중인 macOS 장치의 운영 체제 버전이 아직 지원되지 않는 경우

기술이 빠른 속도로 개발됨에 따라, 사용 중인 장치의 운영 체제 버전을 회사 지원팀이 아직 테스트하지 못했을 수도 있습니다. 즉, 최신 버전의 macOS는 회사의 일부 다른 도구와 연동되지 않을 수도 있습니다.

이 문제를 해결하려면 회사 지원팀에 문의해야 합니다. 연락처 정보는 [회사 포털 웹 사이트](https://portal.manage.microsoft.com#HelpDeskDialog)를 참조하세요.
