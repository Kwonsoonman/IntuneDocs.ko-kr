# <a name="get-started-with-device-compliance-in-intune-azure-preview"></a>Intune Azure 미리 보기에서 장치 준수 시작


[!INCLUDE[azure_preview](../includes/azure_preview.md)]

-   Azure Active Directory 구독

-   iOS

-   Windows 8.1

-   Windows Phone 8.1

-   Windows 10
